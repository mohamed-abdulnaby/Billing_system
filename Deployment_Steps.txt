Status: ✅ DEBUGGED & PUSHED (No Errors Found)

### 💻 Local Development Setup
To run the SvelteKit frontend locally so you can see your changes instantly:

1. Open your terminal and navigate to the frontend folder:
   cd frontend

2. Install the necessary Node packages (Vite, Svelte, Tailwind, etc.):
   npm install

3. Start the development server:
   npm run dev

4. Open your browser to `http://localhost:5173`. Any changes you make to the `.svelte` files will automatically refresh!

### 🏭 Production Deployment
When you are completely finished developing and want to deploy the frontend to a real production server alongside your Java Tomcat backend:

1. Inside the `frontend` folder, run:
   npm run build

2. This will compile your Svelte components into highly optimized vanilla HTML/CSS/JS inside a `.svelte-kit/output` directory.
3. You can then copy these optimized static files directly into your Tomcat `webapp` folder, or host them separately on Vercel or an Nginx server.
